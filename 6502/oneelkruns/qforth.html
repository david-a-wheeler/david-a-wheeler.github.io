QForth 2.4 and QForth 2.4g

QForth 2.4 is an updated version of Toshiyasu Morita's original. QForth is a small integer Forth for Apple IIe/IIc/IIgs computers. It will not work with an Apple II+. Unlike most Apple Forths, QForth uses normal ProDOS text files for source input. It also has ProDOS file support as well as block disk access. Version 2.4 fixes serious bug in +loop and in file loading (thanks J. Matthews!).

Minimum requirements: Apple IIe/IIc/IIc+/IIgs with 128k RAM and one 5.25" floppy drive

These are ShrinkIt archives, change the extension to '.shk' once downloaded!

Known quirks:

  • r@ doesn't work within a do loop.
  • +loop doesn't like negative increments. Use a while loop if you need to count backwards.

Additional Files:

These files add to QForth's functionality. Most recently updated are:
  • SERIAL.4TH -- Serial port words for the Apple IIgs only
  • CHEX.4TH -- For compiling machine code more easily
  • HEX.4TH -- Hex bytes as constants
Read the file comments to see what each file requires.
  • SERIAL.4TH -- Serial port words for Apple IIgs only, modem port only
  • CHEX.4TH -- Hex words for easing compiling of machine code
  • HEX.4TH -- Constants to define hex bytes, helps make up for QForth's lack of hex mode
  • GSTIME.4TH -- Words to read the Apple IIgs clock and implement some simple timers.
  • GRAPHICS.4TH -- Combined lores and hires words.
  • GR.4TH -- Hires graphics words.
  • LOGR.4TH -- Lores graphics words.
  • GRCHAR.4TH -- Hires characters.
  • CMDLINE.4TH -- Words to read the ECP8 command line.
  • DUMP.4TH -- Dump a range of memory.
  • PRINTER.4TH -- Printer control words.
  • DOUBLE.4TH -- 32-bit math words. Replaces SCALE.4TH.
  • STRING.4TH -- Null-terminated string words.
  • TIME.4TH -- Read the date and time from ProDOS.
  • UTIL.4TH -- Various little utility words (2dup, 2swap, etc).
