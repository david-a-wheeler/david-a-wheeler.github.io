TRAX - DOS 3.3 Disk Utilities

TRAX is a collection of DOS 3.3 utility programs including: a sector editor (multiple buffers, fast searches, "online" DOS 3.3 reference), a VTOC editor (mark sectors used/free, trace files on the disk), a hodgepodge of DOS utils (bad sector scan, raw track reads, zap DOS commands, change boot program, etc.) Older versions of similar programs are also on the disk. All are in Applesoft BASIC with machine language components.

Minimum requirements: Apple II+ with 48k RAM **

(** archive is a shrunk 140k disk)
