Sim65 - A 65C02 Microcomputer Simulator

Sim65 simulates a 65C02 based microcomputer with 24k of RAM. It includes documentation and all the files necessary to learn to use the 65C02. A 6502 assembler is also included along with a stripped-down version of QForth. Source for the application and QForth is included.

Minimum requirements: Any Macintosh

Download: sim65.cpt.hqx - Sim65, source and docs. (BinHex, CompactPro archive)
