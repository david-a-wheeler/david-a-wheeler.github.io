Some related pages

Interactive Fiction In The iOS Age: A Text-Based Love Story is an overview of interactive fiction and hints at some iOS specifics. It highly recommends Frotz as a general underpinning implementation and Frotz for iOS for iPhones/iOS.

Andrew Plotkin has posted an iOS framwork for interactive fiction. The iOS framework he uses includes "Full support for VoiceOver (speech output) and dictation (speech input) on devices that offer these features."

This recommends using JFrotz on Android - but note that they use an external PC to get the games themselves (why!?!?).

Sussman recommends running Twisty on Android. It has a few IF game recommendations, including Violet, Bronze, Anchorhead, and also Spider and Web.
