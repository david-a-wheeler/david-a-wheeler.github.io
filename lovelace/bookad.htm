Ada95: The Lovelace Tutorial

Ada95: The Lovelace Tutorial by David A. Wheeler explains the Ada95 computer programming language for those who already know another programming language (such as C, C++, Pascal, or Fortran). The book can be used in classroom settings or for self-study.

Ada95: The Lovelace Tutorial is:

  • Easy to Read. The tutorial is divided into many small sections, each presenting a single key concept. Most code examples are short, making it easier to understand the concept.
  • Easy to Remember. Most sections end in a quiz question, reinforcing the concepts of the section and making it easier to remember the material.
  • Complete. The tutorial covers all of Ada95's key features, including its object-oriented features, tasking, and generics.
  • Practical. Hints on how to apply Ada95 capabilities are spread throughout the tutorial. Sections cover practical details on how to interface to programs written in C and how to structure larger programs.
  • Internet-aware. Sections show how to write Java applets and applications in Ada95, and how to get Ada-related information on the web. Many footnotes refer to web locations for more information.
  • Scaleable. An award-winning text-based role playing game is included. This program shows how to apply various Ada capabilities together in a complete program. An entire chapter is devoted to showing how to develop larger programs by example and explaining this program's components. The book includes several diagrams in Unified Modeling Language (UML) notation to show how the program works.
  • Ready for classroom use Each chapter ends with a set of additional questions, useful in classroom settings.

The book is 292 pages long and was published in March 1997.
