Doodle Applet

Below should be a ``doodle'' applet (if your browser supports the Java Virtual Machine). Just put your mouse in the area below, hold down the mouse button, and doodle. While the mouse button is down, you'll be drawing; release the mouse button to stop drawing. Note that you can put other windows on top and then move them away; this applet remembers what you've done so it can redraw your doodling.

Doodle away!
