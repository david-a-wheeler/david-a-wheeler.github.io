Local Version of Lovelace: An Ada Tutor

Welcome to the local version of Lovelace, an Ada tutorial! This tutorial will explain the basics of the Ada programming language.

It's important to understand that you are looking a local version of Lovelace. That means that you can use this version of the Lovelace tutorial without a connection to the Internet (and even without a modem).

Having a local version of a tutorial implies some limitations:

  1. This tutorial will occasionally refer you to other resources outside this tutorial. Attempting to follow such references will fail unless you have a working connection to the Internet.
  2. You are probably using an older and less complete version of the tutorial. The best way to always use the latest version is to use Lovelace directly through the World Wide Web (WWW).

You can go to the normal Lovelace home page.

You can also:
  • Attempt to connect to the World Wide Web version of Lovelace.
  • See the version information.
